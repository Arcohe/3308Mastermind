gameboard.setscore(score, checknum)¶

Takes the score and sets the pins to indicate correctness. Black is right color and placement. White is right color, wrong placement.

Parameters:
  • score (list.) – List from scoreGuess that will get translated to the graphics.
  • checknum (integer.) – The number of guesses the user has checked.
Returns:

black – number of black pins.
